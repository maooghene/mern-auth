# 🔐 Secure MERN Stack Authentication System

A production-ready full-stack authentication solution. This project demonstrates secure user flows, cloud database integration, and automated email services.

## 🚀 [Live Demo](https://vercel.app)

## ✨ Key Features
*   **Secure Auth**: JWT-based authentication using HTTP-only cookies (XSS Protection).
*   **Email Verification**: Real-time OTP delivery via Nodemailer & Gmail SMTP.
*   **Password Recovery**: Full reset flow with secure token verification.
*   **Responsive UI**: Modern, mobile-first design built with Tailwind CSS.
*   **State Management**: Centralized user data using React Context API.

## 🛠️ Tech Stack
- **Frontend**: React.js (Vite), Tailwind CSS, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas (Mongoose)
- **Deployment**: Vercel

## ⚙️ Setup Instructions
1. Clone the repo: `git clone https://github.com`
2. Install dependencies: 
   - `cd client && npm install`
   - `cd server && npm install`
3. Configure `.env` variables (refer to `.env.example`).
4. Run locally: `npm run dev`
